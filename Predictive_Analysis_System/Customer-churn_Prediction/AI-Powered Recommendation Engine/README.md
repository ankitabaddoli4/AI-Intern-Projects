# AI-Powered Recommendation Engine

## Description
This project is developed as part of an AI Internship.
It implements an AI-powered recommendation engine that suggests movies to users based on content similarity using machine learning techniques.

## Objectives
- Analyze item features (movie genres)
- Build a content-based recommendation model
- Generate personalized movie recommendations

## Features
- Data preparation and processing
- Text vectorization
- Similarity-based recommendation generation
- Display of recommended items

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Project Structure
- recommendation_system.py
- README.md

## How to Run
1. Install required libraries  
2. Run the recommendation system


## Outcome
The system provides personalized movie recommendations based on content similarity using cosine similarity.
